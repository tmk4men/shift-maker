# 01. 法令上の制約

> 原則すべて**ハード制約**。数値は改正されるため設定値として外出しする。
> `✓` = 当初リストにあった項目 / `＋` = 追加項目

## A. 労働時間

| ID | 制約 | 内容 | 備考 |
|---|---|---|---|
| LAW-001 ✓ | 法定労働時間 | 1日8時間・週40時間以内（原則） | 労基法32条 |
| LAW-002 ＋ | 特例事業場の週44時間 | 商業・映画演劇・保健衛生・接客娯楽業で**常時10人未満**の事業場は週44時間 | 小規模飲食店/小売/クリニック等は該当しやすい |
| LAW-003 ＋ | 「1日」「1週」の定義 | 1日＝暦日（0:00〜24:00）、1週＝就業規則で定める起算日から7日（定めなければ日曜起算） | 夜勤の日跨ぎ計算に必須 |
| LAW-004 ＋ | 2暦日にまたがる勤務 | 継続勤務は**始業日の労働**として通算する | 夜勤22:00〜翌7:00は始業日の9時間 |
| LAW-005 ✓ | 36協定なしの時間外禁止 | 36協定未締結／未届出なら法定時間超・法定休日労働は不可 | |
| LAW-006 ＋ | 時間外の上限（原則） | 月45時間・年360時間（1年単位の変形労働時間制は月42時間・年320時間） | |
| LAW-007 ＋ | 時間外の上限（特別条項） | 年720時間以内／休日労働を含め単月100時間未満／2〜6か月平均80時間以内／月45時間超は年6回まで | 4条件すべて同時に満たす必要あり |
| LAW-008 ＋ | 労働時間の通算 | 同一使用者の複数事業場、および**副業・兼業先の労働時間も通算**（労基法38条） | ダブルワーク者の残業判定に影響 |
| LAW-009 ✓ | 変形労働時間制 | 採用時はその制度のルールに従う（下記 LAW-010〜012） | |
| LAW-010 ＋ | 1か月単位の変形 | 期間平均で週40時間以内。**シフトは期間開始前に特定**が必要、原則事後変更不可 | シフト制で最も使われる |
| LAW-011 ＋ | 1年単位の変形 | 1日10時間・週52時間上限、連続勤務は原則6日まで（特定期間は12日まで）、年間労働日数280日以内 | |
| LAW-012 ＋ | 1週間単位の非定型的変形 | 小売・旅館・料理店・飲食店で**30人未満**のみ。1日10時間まで、前週までに書面通知 | |
| LAW-013 ✓ | フレックスタイム制 | コアタイム/フレキシブルタイム、清算期間の総枠内。清算期間1か月超は労基署届出＋各月週平均50時間以内 | |
| LAW-014 ＋ | みなし労働時間制 | 事業場外みなし・裁量労働制の対象者はシフトの組み方が別扱い | |
| LAW-015 ＋ | 管理監督者 | 労働時間・休憩・休日の規定は適用除外。ただし**深夜割増と健康確保義務は適用** | 名ばかり管理職に注意 |

## B. 休憩・休日・インターバル

| ID | 制約 | 内容 | 備考 |
|---|---|---|---|
| LAW-020 ✓ | 休憩（6時間超） | 6時間を超える勤務は45分以上 | |
| LAW-021 ✓ | 休憩（8時間超） | 8時間を超える勤務は1時間以上 | |
| LAW-022 ＋ | 休憩の3原則 | ①労働時間の**途中**に付与 ②原則**一斉**付与（労使協定で除外可／一部業種は適用除外） ③**自由利用** | 休憩中の電話番は休憩にならない |
| LAW-023 ＋ | 休憩の交代要員 | 一斉付与しない場合、休憩中も必要人数を満たす配置が必要 | 実質的にシフト上の要件 |
| LAW-024 ✓ | 法定休日 | 毎週1日以上、または4週間で4日以上 | 4週4日制は就業規則に起算日の定めが必要 |
| LAW-025 ＋ | 法定休日と所定休日の区別 | 週休2日のうち法定休日は1日。割増率が異なる（35% vs 25%） | どちらを法定休日とするか設定必須 |
| LAW-026 ＋ | 休日の単位 | 原則**暦日**（0:00〜24:00の24時間）休ませる | 夜勤明け当日は休日にカウントできない |
| LAW-027 ✓＋ | 勤務間インターバル | 終業から次の始業まで一定時間（例11時間）確保。**現行は努力義務**（労働時間等設定改善法） | 業種によっては義務（IND参照） |

## F. 契約・シフト制特有

| ID | 制約 | 内容 | 備考 |
|---|---|---|---|
| LAW-070 ✓ | 労働契約・就業規則・労使協定の遵守 | 所定労働時間・所定休日・シフトパターンの定め | |
| LAW-071 ＋ | 労働条件明示義務 | 始業終業時刻、就業場所・業務の**変更の範囲**（2024年4月〜）、更新上限の明示 | シフト制は「原則的な始業終業時刻＋シフトによる」明示が必要 |
| LAW-072 ＋ | シフト制の留意事項（厚労省通知） | ①シフトの**作成・変更ルール**を労使で合意 ②シフトの**通知期限**（例：〇日前まで）と通知方法を定める ③一方的な削減は不可 | 2022年「いわゆる『シフト制』により働く労働者の適切な雇用管理を行うための留意事項」 |
| LAW-073 ＋ | 労働日数・時間の目安 | 「週〇日程度」など**最低限度の労働日数・時間**を契約で示すことが望ましい | |
| LAW-074 ＋ | 不利益取扱いの禁止 | 有給取得・育児介護制度利用・妊娠出産等を理由にシフトを減らすのは違法 | **アルゴリズムが無意識に行わないよう要注意** |
| LAW-075 ＋ | 同一労働同一賃金 | 正規・非正規で不合理な待遇差を設けない（シフトの割り振り含む） | |
| LAW-076 ＋ | 派遣労働者 | 派遣元/先の責任分担、事業所単位・個人単位の期間制限（抵触日）、労働者派遣契約の範囲内 | |
| LAW-077 ＋ | 外国人労働者 | 在留資格ごとの就労制限。**留学生は原則週28時間以内**（長期休業中は1日8時間） | 資格外活動許可の管理 |
| LAW-078 ＋ | 高年齢者 | 65歳までの雇用確保措置、70歳までの就業確保努力義務 | |

## G. 社会保険・税（本人の希望に直結）

| ID | 制約 | 内容 | 備考 |
|---|---|---|---|
| LAW-080 ✓＋ | 社会保険の適用要件 | 週の所定労働時間が正社員の3/4以上、または短時間労働者の適用拡大要件（週20時間以上・月額賃金8.8万円以上・2か月超の雇用見込み・学生を除く・企業規模要件） | **企業規模要件は段階的に縮小中。必ず最新確認** |
| LAW-081 ＋ | 雇用保険の適用要件 | 週20時間以上かつ31日以上の雇用見込み | 適用拡大の改正動向あり |
| LAW-082 ✓＋ | 扶養の範囲（税・社保） | 所得税の非課税ライン、住民税のライン、社会保険の被扶養者ライン（月額換算あり） | **いわゆる「年収の壁」は改正が続いているため金額をハードコードしない。設定値で持つ** |
| LAW-083 ＋ | 年間の上限管理 | 扶養内希望者は**年度累計**で管理し、年末に近づくほど厳しく制御する | 月単位だけの管理では破綻する |
