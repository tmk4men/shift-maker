# 03. 業種別の追加制約

> 対象業種を決めたら、該当セクションの制約を有効化する。
> すべて `＋`（当初リストになかった追加項目）。

## 医療（病院・クリニック）

| ID | 内容 |
|---|---|
| IND-001 | 医療法の人員配置標準（医師・看護師・薬剤師の患者数比） |
| IND-002 | 看護配置基準（7対1、10対1 等）を24時間通じて満たす。診療報酬に直結 |
| IND-003 | 夜勤72時間ルール（月平均夜勤時間72時間以内／入院基本料の要件） |
| IND-004 | 三交代（日勤・準夜・深夜）／二交代（日勤・16時間夜勤）のパターン管理 |
| IND-005 | 医師の時間外上限（A/B/C水準、2024年4月〜）と追加的健康確保措置 |
| IND-006 | 医師の**勤務間インターバル9時間以上**（連続勤務28時間制限）— 義務 |
| IND-007 | 宿日直許可の有無で労働時間の扱いが変わる |
| IND-008 | オンコール待機の扱い |
| IND-009 | 診療科・病棟ごとの資格要件（認定看護師、放射線技師 等） |

## 介護・福祉

| ID | 内容 |
|---|---|
| IND-020 | 介護保険法の人員基準（3:1、夜勤者数、常勤換算数）。基準割れは報酬減算 |
| IND-021 | 夜勤職員配置加算・人員配置加算などの算定要件を満たす配置 |
| IND-022 | 管理者・サービス提供責任者・生活相談員・機能訓練指導員の配置と兼務可否 |
| IND-023 | 常勤換算（FTE）の計算 |
| IND-024 | 訪問介護の移動時間・直行直帰、1日の訪問件数上限 |
| IND-025 | 利用者ごとの担当者継続性（できるだけ同じヘルパー） |
| IND-026 | 同性介助の希望への配慮 |

## 保育

| ID | 内容 |
|---|---|
| IND-030 | 年齢別の保育士配置基準（0歳3:1、1・2歳6:1、3歳15:1 等。基準改定あり） |
| IND-031 | 開所時間全体（早朝・延長保育含む）での配置維持 |
| IND-032 | 保育士資格者の割合要件、常時2名以上配置 |
| IND-033 | 行事・お散歩・プール時の増員 |

## 小売・飲食

| ID | 内容 |
|---|---|
| IND-040 | 1週間単位の非定型的変形労働時間制（30人未満）の活用（LAW-012） |
| IND-041 | 特例事業場の週44時間（10人未満）（LAW-002） |
| IND-042 | 発注・棚卸・納品時間帯の担当割当 |
| IND-043 | 現金取扱・レジ締めは有権限者のみ |
| IND-044 | 深夜営業の1人勤務回避（防犯・安全配慮） |
| IND-045 | 酒類提供・調理は有資格者／年齢制限（未成年者の酒類取扱制限） |
